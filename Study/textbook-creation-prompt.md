# 教科書作成プロンプト

以下の構造に従って、指定されたトピックの教科書を作成してください。

## 出力フォルダ構造

```
{教科書名}/
├── phase1_{第1章タイトル}/
│   ├── 01_{節1タイトル}.md
│   ├── 02_{節2タイトル}.md
│   └── ...
├── phase2_{第2章タイトル}/
│   ├── 01_{節1タイトル}.md
│   └── ...
└── ...
```

## 命名規則（必須）

### 章フォルダ
- 形式: `phase{番号}_{タイトル}`
- 番号: 1から始まる連番（1, 2, 3...）
- 例: `phase1_基礎編`, `phase2_実践編`, `phase3_応用編`

### 節ファイル
- 形式: `{2桁番号}_{タイトル}.md`
- 番号: 01から始まる2桁の連番（01, 02, 03...）
- 拡張子: 必ず `.md`
- 例: `01_はじめに.md`, `02_環境構築.md`, `03_基本操作.md`

---

## プロンプトテンプレート

以下をコピーして使用してください：

```
# 教科書作成依頼

## トピック
{ここにトピックを記入}

## 対象読者
{初心者 / 中級者 / 上級者}

## 章構成（案）
1. {第1章のテーマ}
2. {第2章のテーマ}
3. {第3章のテーマ}
...

## 要件
- フォルダ構造: phase{番号}_{章タイトル}/ 配下に 01_{節タイトル}.md 形式
- 各節は1つのトピックに集中し、300〜800語程度
- コードサンプルがある場合はシンタックスハイライト付きコードブロックを使用
- 見出しは # で始める（H1は各ファイルで1つのみ）
- 必要に応じて表、リスト、強調を使用

## 出力形式
各ファイルの内容を以下の形式で出力してください：

---
ファイル: phase1_基礎編/01_はじめに.md
---
（マークダウン内容）
---

```

## 使用例

### 入力例

```
# 教科書作成依頼

## トピック
Pythonプログラミング入門

## 対象読者
初心者

## 章構成（案）
1. Pythonの基礎
2. データ型と変数
3. 制御構文
4. 関数とモジュール

## 要件
- フォルダ構造: phase{番号}_{章タイトル}/ 配下に 01_{節タイトル}.md 形式
- 各節は1つのトピックに集中し、300〜800語程度
- コードサンプルがある場合はシンタックスハイライト付きコードブロックを使用
```

### 出力例

```
---
ファイル: phase1_Pythonの基礎/01_Pythonとは.md
---
# Pythonとは

Pythonは、シンプルで読みやすい構文を持つプログラミング言語です...

## 特徴

- **読みやすさ**: インデントによる構造化
- **汎用性**: Web、AI、データ分析など幅広い用途
- **豊富なライブラリ**: pip で簡単にインストール

## サンプルコード

```python
print("Hello, Python!")
```

---
ファイル: phase1_Pythonの基礎/02_環境構築.md
---
# 環境構築

Pythonをインストールして開発環境を整えましょう...
```

## 注意事項

1. **フォルダ名に使えない文字を避ける**: `/`, `\`, `:`, `*`, `?`, `"`, `<`, `>`, `|`
2. **ファイル名は短く明確に**: 長すぎるタイトルは避ける
3. **番号の一貫性**: 章は1から、節は01から開始
4. **各ファイルは独立して読める**: 前提知識が必要な場合は冒頭で言及
