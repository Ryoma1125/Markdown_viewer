# 認証・認可 (OAuth/JWT)

## 概要

OAuth 2.0、OpenID Connect、JWT を使った認証・認可の仕組みを学ぶカリキュラムです。

## 前提知識

- Python の基本
- HTTP / REST API の基本
- Web アプリケーションの基本

## 学習期間の目安

- 1日1〜2時間: 約8日
- 集中学習: 約5日

## カリキュラム構成

```
Auth-OAuth-JWT/
├── README.md（このファイル）
├── phase1/
│   ├── 01_認証認可とは.md           # 概念、用語
│   └── 02_パスワード認証.md          # 基本的な認証
├── phase2/
│   ├── 01_セッション認証.md          # Cookie/Session
│   └── 02_JWT入門.md                # JWT の仕組み
├── phase3/
│   ├── 01_OAuth2.0基礎.md           # OAuth フロー
│   └── 02_OAuth2.0実装.md           # 実装例
├── phase4/
│   ├── 01_OpenIDConnect.md          # OIDC
│   └── 02_ソーシャルログイン.md       # Google/GitHub
└── phase5/
    └── 01_総仕上げ.md                # 実践プロジェクト
```

## 到達目標

- 認証と認可の違いを説明できる
- JWT を使った認証を実装できる
- OAuth 2.0 フローを理解できる
- ソーシャルログインを実装できる
