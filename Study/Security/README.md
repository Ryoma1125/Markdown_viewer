# セキュリティ基礎

## 概要

Web アプリケーションのセキュリティを体系的に学ぶカリキュラムです。OWASP Top 10 を中心に、実践的な脆弱性対策を習得します。

## 前提知識

- Web 開発基礎（HTTP、HTML、JavaScript）
- Python または他のバックエンド言語
- データベース基礎

## 学習期間の目安

- 1日1〜2時間: 約5日
- 集中学習: 約3日

## カリキュラム構成

```
Security/
├── README.md（このファイル）
├── phase1/
│   ├── 01_セキュリティ概論.md        # OWASP Top 10
│   └── 02_インジェクション.md        # SQL/Command Injection
├── phase2/
│   ├── 01_認証の脆弱性.md            # パスワード、セッション
│   └── 02_XSS_CSRF.md               # クロスサイト攻撃
├── phase3/
│   ├── 01_暗号化とハッシュ.md        # 暗号基礎
│   └── 02_AWSセキュリティ.md        # IAM、WAF
├── phase4/
│   └── 01_総仕上げ.md               # セキュアコーディング
```

## 到達目標

- OWASP Top 10 の脆弱性を理解し対策できる
- セキュアなコードを書ける
- AWS のセキュリティサービスを活用できる
- セキュリティレビューができる
