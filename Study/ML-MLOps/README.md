# 機械学習基礎・MLOps

## 概要

機械学習の基礎からMLOpsまで、実務で必要な知識を学ぶカリキュラムです。あなたのAI開発経験を土台に、より体系的な理解を目指します。

## 前提知識

- Python の基本
- NumPy / Pandas の基本
- AWS SageMaker の経験（あると良い）

## 学習期間の目安

- 1日1〜2時間: 約8日
- 集中学習: 約5日

## カリキュラム構成

```
ML-MLOps/
├── README.md（このファイル）
├── phase1/
│   ├── 01_機械学習入門.md           # 教師あり・なし学習
│   └── 02_データ前処理.md           # 特徴量エンジニアリング
├── phase2/
│   ├── 01_回帰分析.md               # 線形回帰、評価指標
│   └── 02_分類問題.md               # ロジスティック回帰、決定木
├── phase3/
│   ├── 01_モデル評価.md             # 交差検証、ハイパーパラメータ
│   └── 02_アンサンブル学習.md        # Random Forest, XGBoost
├── phase4/
│   ├── 01_ディープラーニング入門.md  # PyTorch基礎

│   ├── 01_MLパイプライン.md         # MLflow, 実験管理

└── phase6/
    └── 01_総仕上げ.md               # 実践プロジェクト
```

## 到達目標

- 機械学習の基本概念を説明できる
- scikit-learn でモデルを構築できる
- モデルを API としてデプロイできる
- MLOps のベストプラクティスを実践できる
