# Redis・キャッシュ戦略

## 概要

Redis の基礎からキャッシュ設計パターンまで学ぶカリキュラムです。パフォーマンス改善の実践的な手法を習得します。

## 前提知識

- Python の基本
- Docker の基本
- データベースの基本

## 学習期間の目安

- 1日1〜2時間: 約7日
- 集中学習: 約4日

## カリキュラム構成

```
Redis-Cache/
├── README.md（このファイル）
├── phase1/
│   ├── 01_Redis入門.md              # 概念、データ型
│   └── 02_Redis基本操作.md          # CLI、基本コマンド
├── phase2/
│   ├── 01_Python連携.md             # redis-py
│   └── 02_キャッシュパターン.md      # Cache-Aside等
├── phase3/
│   ├── 01_セッション管理.md          # セッション、TTL
│   └── 02_高度な機能.md              # Pub/Sub、Lua
└── phase4/
    └── 01_総仕上げ.md                # 実践プロジェクト
```

## 到達目標

- Redis のデータ構造を使い分けられる
- キャッシュ戦略を設計できる
- セッション管理を実装できる
