# WebSocket・リアルタイム通信

## 概要

WebSocket を使ったリアルタイム通信の実装を学ぶカリキュラムです。チャット、通知、ライブ更新などを実装できるようになります。

## 前提知識

- Python / FastAPI の基本
- HTTP の基本
- JavaScript の基本

## 学習期間の目安

- 1日1〜2時間: 約5日
- 集中学習: 約3日

## カリキュラム構成

```
WebSocket-Realtime/
├── README.md（このファイル）
├── phase1/
│   ├── 01_リアルタイム通信とは.md     # HTTP vs WebSocket
│   └── 02_WebSocket基礎.md           # プロトコル、接続
├── phase2/
│   ├── 01_FastAPI実装.md             # サーバー実装
│   └── 02_クライアント実装.md         # フロントエンド
├── phase3/
│   ├── 01_チャット実装.md             # チャットアプリ
│   └── 02_スケーリング.md             # Redis Pub/Sub
└── phase4/
    └── 01_総仕上げ.md                 # 実践プロジェクト
```

## 到達目標

- WebSocket の仕組みを理解できる
- リアルタイムチャットを実装できる
- 通知システムを構築できる
- スケーラブルな設計ができる
