# サーバーレスアーキテクチャ

## 概要

AWS Lambda を中心としたサーバーレスアーキテクチャを深掘りするカリキュラムです。現場で使っている Lambda の知識を体系化し、より高度なパターンを習得します。

## 前提知識

- AWS 基礎（Lambda, S3, API Gateway 経験者）
- Python プログラミング
- REST API の基本

## 学習期間の目安

- 1日1〜2時間: 約5日
- 集中学習: 約3日

## カリキュラム構成

```
Serverless/
├── README.md（このファイル）
├── phase1/
│   ├── 01_サーバーレス概論.md        # なぜサーバーレスか
│   └── 02_Lambda深掘り.md           # コールドスタート対策
├── phase2/
│   ├── 01_イベント駆動設計.md        # EventBridge, SNS, SQS
│   └── 02_Step_Functions.md         # ワークフロー
├── phase3/
│   ├── 01_API設計パターン.md         # REST, GraphQL
│   └── 02_認証とセキュリティ.md      # Cognito, IAM
├── phase4/
│   └── 01_総仕上げ.md               # 実践プロジェクト
```

## 到達目標

- サーバーレスの利点と制約を説明できる
- Lambda の最適化ができる
- イベント駆動アーキテクチャを設計できる
- 本番運用に耐えるサーバーレスシステムを構築できる
