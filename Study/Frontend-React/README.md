# フロントエンド（TypeScript + React）

## 概要

TypeScript と React を使ったモダンフロントエンド開発を基礎から学ぶカリキュラムです。バックエンドエンジニア視点で効率よく習得できるよう構成しています。

## 前提知識

- HTML/CSS の基本（なくても可）
- JavaScript の基本（Python と似てるので大丈夫）
- API の概念

## 学習期間の目安

- 1日1〜2時間: 約3週間
- 集中学習: 約10日

## カリキュラム構成

```
Frontend-React/
├── README.md（このファイル）
├── phase1/
│   ├── 01_TypeScript基礎.md         # 型システム入門
│   └── 02_TypeScript実践.md         # 高度な型
├── phase2/
│   ├── 01_React入門.md              # コンポーネント基礎
│   └── 02_JSX_Props.md              # JSX と Props
├── phase3/
│   ├── 01_State管理.md              # useState
│   └── 02_副作用.md                 # useEffect、API連携
├── phase4/
│   ├── 01_フォーム.md               # フォーム処理
│   └── 02_ルーティング.md           # React Router
├── phase5/
│   ├── 01_状態管理.md               # Context、Zustand
│   └── 02_データフェッチ.md         # TanStack Query
└── phase6/
    └── 01_総仕上げ.md               # 実践プロジェクト
```

## 到達目標

- TypeScript で型安全なコードを書ける
- React でコンポーネントを設計・実装できる
- API と連携したフロントエンドを構築できる
- 状態管理の選択ができる
