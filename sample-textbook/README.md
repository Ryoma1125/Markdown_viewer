# サンプル教科書へようこそ

この教科書は Markdown Textbook Viewer のデモ用サンプルです。

## 学習目標

- Markdown Textbook Viewer の基本的な使い方を理解する
- 教科書の構造と命名規則を学ぶ

## 内容

この教科書は以下の章で構成されています：

### 第1章: 基礎編
- はじめに
- 環境構築

### 第2章: 応用編
- 基本操作
- 応用テクニック

## 使い方

左側のサイドバーから読みたい章・節を選択してください。
キーボードの `←` `→` キーで前後のページに移動できます。
